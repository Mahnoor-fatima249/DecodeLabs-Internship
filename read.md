## 🚀 Overview
This is a deterministic, rule-based AI chatbot built as part of the Phase 1 training. It follows the **IPO Model** (Input-Process-Output) to simulate basic human interaction.



## 🛠️ Technical Features
- **Sanitization:** Uses `.lower().strip()` to normalize user input, making it case-insensitive.
- **Control Flow:** Implemented using robust `if-elif-else` logic gates.
- **Continuous Loop:** Utilizes a `while True` loop to ensure the session stays active.
- **White Box Architecture:** 100% predictable, traceable, and debuggable logic.


## 🏃 How to Run
1. Ensure Python 3.x is installed.
2. Run command: `python3 "AI Chatbot.py"`
3. Libraries: No external libraries needed (Standard Library only).
